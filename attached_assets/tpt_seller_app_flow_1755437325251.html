All‑in‑One TPT Seller App — Clickable Product Flow

Click a box to jump to details. Use this to brief devs and align scope.

Legend: Blue: Core UX Green: SEO/Content Yellow: Tracking/Jobs Pink: Social

Sign up / Login

  • Email + password or Google OAuth.
  • Onboarding: choose goals (traffic, conversion, social growth) to personalize tips.

Connect TPT Account

  • Method A: API (if available).
  • Method B: Guided import (CSV/HTML) + optional headless browser for scrape sync.
  • Token vault for credentials; read‑only by default.

Import Products

  • Pull: title, description, tags, price, preview images, last updated.
  • Normalize + version each product to support A/B tests.

Dashboard

  • Tiles: SEO tasks, rank alerts, social queue, recent wins.
  • Search + filters by subject/grade/format.

SEO Optimizer

  • Keyword ideas via Suggest + internal search scraping.
  • AI rewrite for titles/descriptions with constraints (length, keywords, tone).
  • SEO score with rationale and quick‑fix checklist.

Keyword & Rank Tracker

  • Track chosen keywords per product; fetch rankings daily/weekly.
  • Notify on drops or breakthroughs; annotate changes.

Social Content Generator

  • Generate Pin titles/descriptions, IG/FB captions.
  • Canva/Cloudinary templates for images; schedule via Buffer/Later.

Analytics & Insights

  • Blend sales/views (manual CSV or API) with rank and SEO edits.
  • Show impact: “After title change +15% CVR”.

Publish Changes

  • One‑click export back to TPT (or copy/paste assistant with change log).
  • Versioning + rollback.

Background Jobs

  • Queues for rank fetching, keyword refresh, report emails.
  • Rate‑limit aware; retries; observability.

Schedule & Queue

  • Calendar for social posts and A/B tests.
  • UTM tagging for attribution.

Settings

  • Integrations: Pinterest, Buffer, MailerLite, Google Sheets.
  • Billing via Stripe; plan limits (products, tracked keywords, AI tokens).
  • Team seats & roles (optional).

Data & Infra

  • PostgreSQL/Supabase; S3‑compatible storage for assets.
  • Auth with JWT; Stripe subscriptions; audit logs.
  • Infra: Vercel/Netlify (frontend), Fly.io/Render (backend) or Supabase Edge Functions.
